# React-Redux-Auth0Lock Boilerplate

```
> git clone https://github.com/Ahrimaan/React-Redux-Auth0-Boilderplate.git
> cd React-Redux-Auth0-Boilderplate
> npm install
> go to src/services/authenticationService
> Add your ClientID and Domian into the lock constructor (described here https://auth0.com/docs/libraries/lock/v10)
> Change the options (described here : https://auth0.com/docs/libraries/lock/v10/configuration )
> Watch out for correct scope frommauth0 to get all userinfos you need
> npm start
```
